# 数据集采集与标注指南 — Stage 1

## 概述

本阶段使用 YOLOv8n 目标检测，检测 4 类目标：
- `0: person` — 学习者的主体
- `1: phone` — 手机（亮屏/黑屏/手持/桌面）
- `2: book` — 书本、笔记本、教材、纸质资料
- `3: laptop` — 笔记本电脑、电脑屏幕（平板电脑可暂时归入此类）

## 采集设备

使用 CanMV K230D 摄像头直接拍摄，或使用手机/相机拍摄后统一缩放到合理分辨率。
推荐采集分辨率：640x360 或更高，保持与部署时一致。

## 采集场景清单

按以下 10 类场景采集，每类尽量覆盖 50-200 张：

### 1. 正常看书 (person + book)
- 人坐在桌前，面前摊开书本
- 不同姿势：低头看书、抬头思考、翻页

### 2. 正常看电脑 (person + laptop)
- 人面对笔记本电脑
- 不同姿势：打字、看屏幕、鼠标操作

### 3. 手机放在桌上但未使用 (person + phone + book/laptop)
- 手机放在桌面一角，人正在学习
- 手机可能亮屏或黑屏

### 4. 手持手机使用 (person + phone)
- 人手持手机在看
- 单手拿手机、双手拿手机、手机放在桌上低头看
- **这是 phone 类别的关键场景，必须充分覆盖**

### 5. 人离开座位 (book/laptop/phone 存在但无 person)
- 空椅子、桌上有学习物品但无人
- 模拟中途离开的场景

### 6. 空桌面
- 完全没有人的桌面
- 可能有书本或电脑，也可能什么都没有

### 7. 不同距离
- 近距离：摄像头距人约 30-50cm（桌面摄像头典型距离）
- 中距离：约 50-80cm
- 稍远距离：约 80-120cm（能看到整个桌面）

### 8. 不同角度
- 正面：摄像头正对人
- 侧面：从侧方拍摄
- 俯视：从上方斜向下拍摄（模拟放在显示器上方的摄像头）
- 斜侧面：45 度角拍摄

### 9. 不同光照
- 正常室内光（日光灯）
- 弱光（台灯、傍晚）
- 强光（白天窗边）
- 背光（人背对窗户）

### 10. 不同背景
- 宿舍桌
- 书桌
- 教室桌
- 自习室桌

## 标注规范

### 工具推荐
- [LabelImg](https://github.com/HumanSignal/labelImg) — YOLO 格式
- [Label Studio](https://labelstud.io/) — 支持导出 YOLO 格式
- [Roboflow](https://roboflow.com/) — 在线标注和管理

### 标注格式
每个图片对应一个同名 `.txt` 文件，每行一个目标：
```
<class_id> <x_center> <y_center> <width> <height>
```
所有坐标都是归一化值 (0~1)。

### 标注原则
1. 矩形框尽量贴合目标边缘
2. 被遮挡但仍可识别主要特征的目标可以标注
3. 极小（小于 20x20 像素）、严重模糊、无法判断类别的目标不标注
4. 同一目标只标注一个框
5. **phone 类别是关键**：
   - 亮屏手机要标
   - 黑屏手机要标（能看出是手机的轮廓）
   - 手持手机要标
   - 桌面手机要标
   - 倾斜/倒扣的手机如果可以判断也要标
6. **person 类别**：
   - 只标注主要学习者
   - 不标注海报里的人、照片里的人、屏幕里的人
   - 不需要全身完整，上半身能判断为"人"即可
7. **laptop 类别**：
   - 优先标注屏幕区域（特征最明显）
   - 如果屏幕看不清，标注整机轮廓
   - 平板电脑暂时归入此类
   - 保持标注方式统一

## 数据集划分

使用 `dataset_tools/split_dataset.py` 按 70/20/10 划分 train/val/test。

## 数据增强（训练时自动）

YOLOv8 训练时的默认增强包括：
- 随机翻转 (horizontal flip)
- 随机缩放 (scale)
- 随机裁剪 (mosaic)
- 颜色抖动

不需要手动做数据增强。

## 第一版数据量建议

- **最少** 500 张（每类场景至少 50 张）
- **推荐** 1000-2000 张
- 如果 phone 类别检测效果不好，重点增加 phone 相关场景

## 质量检查

采集和标注完成后，运行 `dataset_tools/check_yolo_labels.py` 检查：
- 标注文件是否存在
- 类别 ID 是否在 0-3 范围内
- 坐标是否在 0-1 范围内
- 宽高是否大于 0
